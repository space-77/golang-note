【2】指针变量： 

  1. package main
  2. import(
  3.         "fmt"
  4. )
  6. func main(){
  7.         var age int = 18
  8.         //&符号+变量 就可以获取这个变量内存的地址
  9.         fmt.Println(&age) //0xc0000a2058
  11.         //定义一个指针变量：
  12.         //var代表要声明一个变量
  13.         //ptr 指针变量的名字
  14.         //ptr对应的类型是：*int 是一个指针类型 （可以理解为 指向int类型的指针）
  15.         //&age就是一个地址，是ptr变量的具体的值
  16.         var ptr *int = &age
  17.         fmt.Println(ptr)
  18.         fmt.Println("ptr本身这个存储空间的地址为：",&ptr)
  20.         //想获取ptr这个指针或者这个地址指向的那个数据：
  21.         fmt.Printf("ptr指向的数值为：%v",*ptr) //ptr指向的数值为：18
  22. }
内存：

总结：最重要的就是两个符号：
1.& 取内存地址
2.* 根据地址取值
