【2】有符号整数类型：
PS:127怎么算出来的？
01111111 -->二进制 ---》转为十进制：
   1*2^6   +   1*2^5  +  1*2^4  +   1*2^3  +   1*2^2  +   1*2^1  +    1*2^0
 = 64      +   32         +    16          +   8        +      4       +    2     +   1
= 127


PS：-128怎么算出来的？
10000000 --->二进制 --->一看就是个负数 

        10000000 --》负数的二进制
减1：01111111
取反：10000000     ---》得到一个正数    2^7 = 128
加负号：-128

【5】这么多整数类型，使用的时候该如何选择呢？
Golang程序中整型变量在使用时,遵守保小不保大的原则,
即:在保证程序正确运行下,尽量使用占用空间小的数据类型
