【1】结构体是用户单独定义的类型，和其它类型进行转换时需要有完全相同的字段(名字、个数和类型) 

  1. package main
  2. import "fmt"
  3. type Student struct {
  4.         Age int
  5. }
  7. type Person struct {
  8.         Age int
  9. }
  11. func main(){
  12.         var s Student = Student{10}
  13.         var p Person = Person{10}
  14.         s = Student(p)
  15.         fmt.Println(s)
  16.         fmt.Println(p)
  17. }
【2】结构体进行type重新定义(相当于取别名)，Golang认为是新的数据类型，但是相互间可以强转 

  1. package main
  2. import "fmt"
  3. type Student struct {
  4.         Age int
  5. }
  7. type Stu Student
  9. func main(){
  10.         var s1 Student = Student{19}
  11.         var s2 Stu = Stu{19}
  12.         s1 = Student(s2)
  13.         fmt.Println(s1)
  14.         fmt.Println(s2)
  16. }
