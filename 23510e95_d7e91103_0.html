【1】绑定指定类型：
方法：需要绑定指定数据类型
函数：不需要绑定数据类型

【2】调用方式不一样：
函数的调用方式:
                  函数名(实参列表)
方法的调用方式：变量.方法名(实参列表) 

  1. package main
  2. import "fmt"
  5. type Student struct{
  6.         Name string
  7. }
  9. //定义方法：
  10. func (s Student) test01(){
  11.         fmt.Println(s.Name)
  12. }
  14. //定义函数：
  15. func method01(s Student){
  16.         fmt.Println(s.Name)
  17. }
  19. func main(){
  20.         //调用函数：
  21.         var s Student = Student{"丽丽"}
  22.         method01(s)
  24.         //方法调用：
  25.         s.test01()
  26. }


【3】对于函数来说，参数类型对应是什么就要传入什么。 


  1. package main
  2. import "fmt"
  5. type Student struct{
  6.         Name string
  7. }
  10. //定义函数：
  11. func method01(s Student){
  12.         fmt.Println(s.Name)
  13. }
  16. func method02(s *Student){
  17.         fmt.Println((*s).Name)
  18. }
  20. func main(){
  21.         var s Student = Student{"丽丽"}
  22.         method01(s)
  23.         //method01(&s)错误
  24.         method02(&s)
  25.         //method02(s)错误
  26. }



【4】对于方法来说，接收者为值类型，可以传入指针类型，接受者为指针类型，可以传入值类型。 

  1. package main
  2. import "fmt"
  5. type Student struct{
  6.         Name string
  7. }
  9. //定义方法：
  10. func (s Student) test01(){
  11.         fmt.Println(s.Name)
  12. }
  14. func (s *Student) test02(){
  15.         fmt.Println((*s).Name)
  16. }
  17. func main(){
  18.         var s Student = Student{"丽丽"}
  19.         s.test01()
  20.         (&s).test01()//虽然用指针类型调用，但是传递还是按照值传递的形式
  22.         (&s).test02()
  23.         s.test02()//等价
  24. }
