【1】结构体、结构体对象的引入：
案例：老师结构体
后续实践中按照自己的需求定义结构体即可

【2】代码： 

  1. package main
  2. import "fmt"
  4. //定义老师结构体，将老师中的各个属性  统一放入结构体中管理：
  5. type Teacher struct{
  6.         //变量名字大写外界可以访问这个属性
  7.         Name string
  8.         Age int
  9.         School string
  10. }
  12. func main(){
  13.         //创建老师结构体的实例、对象、变量：
  14.         var t1 Teacher // var a int
  15.         fmt.Println(t1) //在未赋值时默认值：{ 0 }
  16.         t1.Name = "马士兵"
  17.         t1.Age = 45
  18.         t1.School = "清华大学"
  19.         fmt.Println(t1)
  20.         fmt.Println(t1.Age + 10)
  21. }
