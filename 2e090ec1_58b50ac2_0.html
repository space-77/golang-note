【1】继承的引入：
当多个结构体存在相同的属性(字段)和方法时,可以从这些结构体中抽象出结构体,在该结构体中定义这些相同的属性和方法，其它的结构体不需要重新定义这些属性和方法，只需嵌套一个匿名结构体即可。也就是说:在Golang中，如果一个struct嵌套了另一个匿名结构体，那么这个结构体可以直接访问匿名结构体的字段和方法，从而实现了继承特性。
【2】代码引入： 


  1. package main
  2. import (
  3.         "fmt"
  4. )
  7. //定义动物结构体：
  8. type Animal struct{
  9.         Age int
  10.         Weight float32
  11. }
  13. //给Animal绑定方法：喊叫：
  14. func (an *Animal) Shout(){
  15.         fmt.Println("我可以大声喊叫")
  16. }
  18. ////给Animal绑定方法：自我展示：
  19. func (an *Animal) ShowInfo(){
  20.         fmt.Printf("动物的年龄是：%v,动物的体重是：%v",an.Age,an.Weight)
  21. }
  23. //定义结构体：Cat
  24. type Cat struct{
  25.         //为了复用性，体现继承思维，嵌入匿名结构体：——》将Animal中的字段和方法都达到复用
  26.         Animal
  27. }
  29. //对Cat绑定特有的方法：
  30. func (c *Cat) scratch(){
  31.         fmt.Println("我是小猫，我可以挠人")
  32. }
  35. func main(){
  36.         //创建Cat结构体示例：
  37.         cat := &Cat{}
  38.         cat.Animal.Age = 3
  39.         cat.Animal.Weight = 10.6
  40.         cat.Animal.Shout()
  41.         cat.Animal.ShowInfo()
  42.         cat.scratch()
  43. }


【3】继承的优点：
提高代码的复用性、扩展性
