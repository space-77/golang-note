【2】方法的声明和调用格式：
声明： 
  1. type A struct {
  2.     Num int
  3. }
  4. func (a A) test() {
  5.     fmt.Println(a.Num)
  6. }
调用：
var a A
a.test()

（1）func (a A) test()相当于A结构体有一个方法叫test
（2）(a A)体现方法test和结构体A绑定关系
