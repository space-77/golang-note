【1】介绍：
字符串就是一串固定长度的字符连接起来的字符序列。


【2】字符串的使用：   


  1. package main
  2. import "fmt"
  4. func main(){
  5.         //1.定义一个字符串：
  6.         var s1 string = "你好全面拥抱Golang"

  11.         //s2[0] = 't'

  17.         //（2）如果字符串中有特殊字符，字符串的表示形式用反引号 ``
  18.         var s4 string = `
  19.         package main
  20.         import "fmt"
  21.         
  22.         func main(){

  28.                 fmt.Println(flag02)
  29.         
  30.                 var flag03 bool = 5 < 9

  33.         `
  34.         fmt.Println(s4)
  36.         //4.字符串的拼接效果：

  43.          "def"+ "abc" + "def" + "abc" + "def"+ "abc" + "def" + "abc" + "def"+
