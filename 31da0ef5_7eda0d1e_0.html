【1】结构体可以使用嵌套匿名结构体所有的字段和方法，即:首字母大写或者小写的字段、方法，都可以使用。 

  1. package main
  2. import (
  3.         "fmt"
  4. )
  7. //定义动物结构体：
  8. type Animal struct{
  9.         Age int
  10.         weight float32
  11. }
  13. //给Animal绑定方法：喊叫：
  14. func (an *Animal) Shout(){
  15.         fmt.Println("我可以大声喊叫")
  16. }
  18. ////给Animal绑定方法：自我展示：
  19. func (an *Animal) showInfo(){
  20.         fmt.Printf("动物的年龄是：%v,动物的体重是：%v",an.Age,an.weight)
  21. }
  23. //定义结构体：Cat
  24. type Cat struct{
  25.         //为了复用性，体现继承思维，嵌入匿名结构体：——》将Animal中的字段和方法都达到复用
  26.         Animal
  27. }
  29. //对Cat绑定特有的方法：
  30. func (c *Cat) scratch(){
  31.         fmt.Println("我是小猫，我可以挠人")
  32. }
  35. func main(){
  36.         //创建Cat结构体示例：
  37.         cat := &Cat{}
  38.         cat.Animal.Age = 3
  39.         cat.Animal.weight = 10.6
  40.         cat.Animal.Shout()
  41.         cat.Animal.showInfo()
  42.         cat.scratch()
  43. }

【3】当结构体和匿名结构体有相同的字段或者方法时，编译器采用就近访问原则访问，如希望访问匿名结构体的字段和方法，可以通过匿名结构体名来区分。 


  1. package main
  2. import (
  3.         "fmt"
  4. )
  7. //定义动物结构体：
  8. type Animal struct{
  9.         Age int
  10.         weight float32
  11. }
  13. //给Animal绑定方法：喊叫：
  14. func (an *Animal) Shout(){
  15.         fmt.Println("我可以大声喊叫")
  16. }
  18. ////给Animal绑定方法：自我展示：
  19. func (an *Animal) showInfo(){
  20.         fmt.Printf("动物的年龄是：%v,动物的体重是：%v",an.Age,an.weight)
  21. }
  23. //定义结构体：Cat
  24. type Cat struct{
  25.         //为了复用性，体现继承思维，嵌入匿名结构体：——》将Animal中的字段和方法都达到复用
  26.         Animal
  27.         Age int
  28. }
  30. func (c *Cat) showInfo(){
  31.         fmt.Printf("~~~~~~~~动物的年龄是：%v,动物的体重是：%v",c.Age,c.weight)
  32. }
  34. //对Cat绑定特有的方法：
  35. func (c *Cat) scratch(){
  36.         fmt.Println("我是小猫，我可以挠人")
  37. }
  40. func main(){
  41.         //创建Cat结构体示例：
  42.         // cat := &Cat{}
  43.         // cat.Age = 3
  44.         // cat.weight = 10.6
  45.         // cat.Shout()
  46.         // cat.showInfo()
  47.         // cat.scratch()
  49.         cat := &Cat{}
  50.         cat.weight = 9.4
  51.         cat.Age = 10 //就近原则
  52.         cat.Animal.Age = 20
  53.         cat.showInfo()//就近原则
  54.         cat.Animal.showInfo()
  55. }
【4】Golang中支持多继承：如一个结构体嵌套了多个匿名结构体，那么该结构体可以直接访问嵌套的匿名结构体的字段和方法，从而实现了多重继承。为了保证代码的简洁性，建议大家尽量不使用多重继承，很多语言就将多重继承去除了，但是Go中保留了。 

  1. package main
  2. import (
  3.         "fmt"
  4. )
  7. type A struct{
  8.         a int
  9.         b string
  10. }
  12. type B struct{
  13.         c int
  14.         d string
  15. }
  17. type C struct{
  18.         A
  19.         B
  20. }
  22. func main(){
  23.         //构建C结构体实例：
  24.         c := C{A{10,"aaa"},B{20,"ccc"}}
  25.         fmt.Println(c)
  26. }
【5】如嵌入的匿名结构体有相同的字段名或者方法名，则在访问时，需要通过匿名结构体类型名来区分。 

  1. package main
  2. import (
  3.         "fmt"
  4. )
  7. type A struct{
  8.         a int
  9.         b string
  10. }
  12. type B struct{
  13.         c int
  14.         d string
  15.         a int
  16. }
  18. type C struct{
  19.         A
  20.         B
  21. }
  23. func main(){
  24.         //构建C结构体实例：
  25.         c := C{A{10,"aaa"},B{20,"ccc",50}}
  26.         fmt.Println(c.b)
  27.         fmt.Println(c.d)
  28.         fmt.Println(c.A.a)
  29.         fmt.Println(c.B.a)
  30. }
