【1】格式灵活： 

  1. package main
  2. import "fmt"
  3. func main(){
  4.         i := 1 //变量的初始化
  5.         for i<=5 {//条件表达式。判断条件
  6.                 fmt.Println("你好 Golang")//循环体
  7.                 i++//迭代
  8.         }
  9. }
【2】死循环： 

  1. package main
  2. import "fmt"
  3. func main(){
  4.         //死循环：
  5.         // for {
  6.         // 	fmt.Println("你好 Golang")
  7.         // }
  9.         for ;; {
  10.                 fmt.Println("你好 Golang")
  11.         }
  12. }
在控制台中结束死循环：ctrl+c
