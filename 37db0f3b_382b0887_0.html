【1】Go支持匿名函数，如果我们某个函数只是希望使用一次，可以考虑使用匿名函数
【2】匿名函数使用方式：
（1）在定义匿名函数时就直接调用，这种方式匿名函数只能调用一次（用的多）
（2）将匿名函数赋给一个变量(该变量就是函数变量了)，再通过该变量来调用匿名函数（用的少）
【3】如何让一个匿名函数，可以在整个程序中有效呢?将匿名函数给一个全局变量就可以了 

  1. package main
  2. import "fmt"
  4. var Func01 = func (num1 int,num2 int) int{
  5.         return num1 * num2
  6. }
  8. func main(){
  9.         //定义匿名函数：定义的同时调用
  10.         result := func (num1 int,num2 int) int{
  11.                 return num1 + num2
  12.         }(10,20)
  14.         fmt.Println(result)
  17.         //将匿名函数赋给一个变量，这个变量实际就是函数类型的变量
  18.         //sub等价于匿名函数
  19.         sub := func (num1 int,num2 int) int{
  20.                 return num1 - num2
  21.         }
  23.         //直接调用sub就是调用这个匿名函数了
  24.         result01 := sub(30,70)
  25.         fmt.Println(result01)
  27.         result02 := sub(30,70)
  28.         fmt.Println(result02)
  30.         result03 := Func01(3,4)
  31.         fmt.Println(result03)
  32. }
