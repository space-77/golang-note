【1】基本语法
if 条件表达式 {
  逻辑代码
}
当条件表达式为ture时，就会执行得的代码。
PS：条件表达式左右的()可以不写，也建议不写 
PS：if和表达式中间，一定要有空格
PS：在Golang中，{}是必须有的,就算你只写一行代码。
