【1】接口本身不能创建实例，但是可以指向一个实现了该接口的自定义类型的变量。

【2】只要是自定义数据类型，就可以实现接口，不仅仅是结构体类型。

【3】一个自定义类型可以实现多个接口 

  1. package main
  3. import "fmt"
  5. type AInterface interface{
  6.         a()
  7. }
  9. type BInterface interface{
  10.         b()
  11. }
  13. type Stu struct{
  15. }
  17. func (s Stu) a(){
  18.         fmt.Println("aaaa")
  19. }
  21. func (s Stu) b(){
  22.         fmt.Println("bbbb")
  23. }
  26. func main(){
  27.         var s Stu
  28.         var a AInterface = s
  29.     var b BInterface = s
  30.         a.a()
  31.         b.b()
  32. }

【4】一个接口(比如A接口)可以继承多个别的接口(比如B,C接口)，这时如果要实现A接口,也必须将B,C接口的方法也全部实现。 

  1. package main
  3. import "fmt"
  5. type CInterface interface{
  6.         c()
  7. }
  9. type BInterface interface{
  10.         b()
  11. }
  13. type AInterface interface{
  14.         BInterface
  15.         CInterface
  16.         a()
  17. }
  19. type Stu struct{
  21. }
  23. func (s Stu) a(){
  24.         fmt.Println("a")
  25. }
  26. func (s Stu) b(){
  27.         fmt.Println("b")
  28. }
  29. func (s Stu) c(){
  30.         fmt.Println("c")
  31. }
  33. func main(){
  34.         var s Stu
  35.         var a AInterface = s
  36.         a.a()
  37.         a.b()
  38.         a.c()
  40. }

【5】interface类型默认是一个指针(引用类型)，如果没有对interface初始化就使用,那么会输出nil


【6】空接口没有任何方法,所以可以理解为所有类型都实现了空接口，也可以理解为我们可以把任何一个变量赋给空接口。
